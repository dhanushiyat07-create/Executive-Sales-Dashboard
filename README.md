# Executive-Sales-Dashboard
Interactive Sales Dashboard created using Microsoft Excel to analyze sales performance, profitability, and business trends.

📌 <h1>Project Overview</h1>

The Executive Sales Dashboard is an interactive business intelligence solution developed using Microsoft Excel to analyze sales performance and generate actionable business insights.

This project transforms raw sales data into meaningful visual reports, enabling stakeholders to monitor key performance indicators (KPIs), identify trends, and support data-driven decision-making.

---
🎯 Business Objective

The primary objectives of this project are:

- Analyze sales performance across multiple regions
- Identify profitable and underperforming product categories
- Monitor business KPIs and sales trends
- Evaluate regional and state-wise performance
- Generate insights to support strategic business decisions

---

📂 Dataset Information

The dataset contains approximately 9,994 sales records with the following attributes:

- Order Information
- Customer Details
- Product Categories
- Regional Data
- Sales Amount
- Profit
- Quantity Sold

---

📊 Key Performance Indicators (KPIs)

The dashboard tracks:

✅ Total Sales

✅ Total Profit

✅ Total Orders

✅ Category Performance

✅ Regional Performance

✅ State-wise Analysis
